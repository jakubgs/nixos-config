# Description

Configuration for [Radxa Rock 5B](https://radxa.com/rock5b/).

# Hardware

* Rockchip RK3588 
* Quad Cortex®-A76 @ 2.2~2.4GHz
* Quad Cortex®-A55 @ 1.8GHz 
* Mali G610MC4 GPU, OpenGL ES1.1, ES2.0, ES3.2 
* 16 GB 64bit LPDDR4x 
* 16MB SPI Flash for bootloader
* eMMC socket
