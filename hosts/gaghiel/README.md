# Description

Configuration for [Radxa Rock 5C](https://radxa.com/products/rock5/5c/), serial number `d92ee05044f80afa`.

# Hardware

* Rockchip RK3588s2
* Quad Cortex®-A76 @ 2.2~2.4GHz
* Quad Cortex®-A55 @ 1.8GHz
* Mali G610MC4 GPU, OpenGL ES1.1, ES2.0, ES3.2
* 8 GB 64bit LPDDR4x
* 64 GB eMMC
